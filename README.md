# PriyaMobileCare
A business website built for "Priya Mobile Care" to showcase mobile repair services, accessories, and online government-related services in a simple and professional way.
Priya Mobile Care is a responsive web page designed to provide customers with clear information about available mobile repair services and online assistance. The website helps users easily explore services, contact the shop, and access important information through a clean and user-friendly interface.

Highlights :
1. Business-focused and informative layout
2. Mobile repair and accessories service showcase
3. Online government service information
4. Responsive design for all devices
5. Easy contact and enquiry options

Tech Stack Used :
1. HTML
2. CSS
3. JavaScript

Purpose of the Project :
1. Create an online presence for a local mobile service shop
2. Display services in a clear and organized manner
3. Make it easy for customers to contact the business
4. Apply frontend web development skills in a real project
